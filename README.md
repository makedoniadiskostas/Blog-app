# Laravel 12 Blog Application

A full-featured blog application built with **Laravel 12** to showcase modern Laravel development practices. The project includes authentication, post management, tags, comments, search functionality, and an admin dashboard with clean MVC architecture.

## Features

### Public Features
- View all blog posts
- Read single blog posts
- Search posts in real time
- Browse posts by tags
- Comment system
- Responsive frontend layout

### Admin / Auth Features
- User registration & login
- Email verification
- Dashboard for managing content
- Create, edit, and delete blog posts
- Create and manage tags
- Authorization for post editing/deleting
- Profile & password settings

## Built With

- Laravel 12
- PHP 8+
- MySQL / MariaDB
- Blade Templates
- Livewire
- Eloquent ORM
- MVC Architecture

## Advanced Concepts Used

- Repository Pattern
- Form Request Validation
- Events & Listeners
- Route Model Binding
- Authorization Policies / Gates
- Dependency Injection
